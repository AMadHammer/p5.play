    Rendering Module

    Defined in: __temp_examples/lib/p5.js:13996

    Base class for all elements added to a sketch, including canvas, graphics buffers, and other HTML elements. Methods in blue are included in the core functionality, methods in brown are added with the p5.dom library. It is not called directly, but p5.Element objects are created by calling createCanvas, createGraphics, or in the p5.dom library, createDiv, createImg, createInput, etc.

    This module provides the following classes:

    • p5.Element
    • p5.Graphics
    • p5.Renderer
