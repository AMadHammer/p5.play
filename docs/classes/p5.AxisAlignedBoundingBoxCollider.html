    p5.AxisAlignedBoundingBoxCollider

    Extends p5.CollisionShape
    Module: p5.play
    Parent Module: p5.play

    An Axis-Aligned Bounding Box (AABB) collision shape, used to detect overlap and compute minimum displacement vectors with other collision shapes.

    Cannot be rotated - hence the name. You might use this in place of an OBB because it simplifies some of the math and may improve performance.
